# Skills

Project-scoped agent skills, scoped to this repo only (nothing installed at the
user level).

## mattpocock/skills

The 17 skills declared in the upstream plugin manifest (`engineering/` +
`productivity/` sets), from [mattpocock/skills](https://github.com/mattpocock/skills)
(MIT — see `LICENSE.upstream`). Deprecated, in-progress, misc, and personal
skills were intentionally excluded. Copied manually (not via the `npx skills`
installer). To refresh, re-copy the matching folders from upstream `skills/`.

## chrome-extensions

Official Chrome MV3 extension guidance + Web Store reference docs, from
[GoogleChrome/modern-web-guidance-src](https://github.com/GoogleChrome/modern-web-guidance-src)
(`skills-src/chrome-extensions`, Apache-2.0). Directly relevant to building the
Linktrail extension. Copied with its full `references/` tree intact.

## next-slice

Repo-local skill (authored here, not vendored) — implements the `docs/issues/`
slices test-first. See its `SKILL.md`.
