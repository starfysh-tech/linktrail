# Skills

Project-scoped agent skills vendored from
[mattpocock/skills](https://github.com/mattpocock/skills) (MIT — see
`LICENSE.upstream`).

These are the 17 skills declared in the upstream plugin manifest
(`engineering/` + `productivity/` sets). Deprecated, in-progress, misc, and
personal skills were intentionally excluded.

Copied manually (not via the `npx skills` installer) and scoped to this repo
only — nothing was installed at the user level.

To refresh, re-copy the matching folders from upstream `skills/`.
